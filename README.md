[ES] Asistente virtual por voz desarrollado en Python e integrado con un flujo de automatización en n8n.  
[EN] Voice-based virtual assistant built with Python and integrated with an n8n automation workflow.

---

## 🚀 Project Overview / Descripción del proyecto

**Jarvis AI Voice Assistant** is a personal project designed to simulate a real-time AI assistant that can interact with the user through voice, process requests and connect with external automation workflows.

**Jarvis AI Voice Assistant** es un proyecto personal diseñado para simular un asistente de IA en tiempo real capaz de interactuar con el usuario mediante voz, procesar peticiones y conectarse con automatizaciones externas.

The goal of this project is to explore how AI agents, voice interaction, APIs and automation platforms can work together to create useful assistant-like experiences.

El objetivo del proyecto es explorar cómo los agentes de IA, la interacción por voz, las APIs y las plataformas de automatización pueden trabajar juntas para crear experiencias similares a un asistente inteligente.
